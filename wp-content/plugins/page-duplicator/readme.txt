== Description ==

Page Duplicator allow you to create duplicate any Post / Page of on your site. That includes posts, pages and custom post types.

== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `gd-post-duplicate` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
